        HideShow
        per_page
        number (optional) Default: 15 Example: 15

        The number of results per page.

        page
        number (optional) Default: 1 Example: 1

        The current page number of results.

        status
        enum (optional) 

        The status of Project (publish,draft,pending).

        category
        number (optional) 

        Filter Projects by Project category Id.
